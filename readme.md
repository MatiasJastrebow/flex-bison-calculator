# Calculadora utilizando Flex y Bison
La calculadora permite crear variables y ejecutar funciones basicas.
Para salir de la calculadora escribir *exit*

## Como correr la calculadora
```
$ make
$ ./calc
```

## Funcionalidades Basicas

### Operaciones 
```
+
-
*
/
^
```

### Crear variables
```
var <name>
var <name> = <value>
```

### Funciones
```
sin(x)
cos(x)
tan(x)
asin(x)
acos(x)
atan(x)
ln(x)
exp(x)
sqrt(x)
```
